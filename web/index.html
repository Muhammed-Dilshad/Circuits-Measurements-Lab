Instructions

Standardization

  1. Set LVDT arm at 0 position:

    2. Adjust the screw gauge to read 10mm. Vary the excitation amplitude adjustment amplifier and set the display value as zero.

  2. Set LVDT arm at +10:

    3. Adjust the screw gauge to read 0mm. Note the display reading and correct it if it is not correct.

  3. Set LVDT arm at -10:

    4. Adjust the screw gauge to read 20mm. If the display reading is not -10, correct it by varying the excitation amplitude adjustment amplifier.

Disclaimer: Standardization is Recommended for Accurate Readings

Measurement

  1. Set LVDT arm at 0:

    2. Vary the arm position in steps using the screw gauge. In each case, note down the display readings and LVDT output voltage measured by using a multimeter.
